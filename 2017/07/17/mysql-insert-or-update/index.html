MySQL 插入时如果存在则更新

发表于 2017-07-17 | 分类于 术业有专攻

SQL

注意

  • 要先添加唯一索引，MySQL根据唯一索引来判断是否重复

优点

  • 效率高，判断当存在重复值时直接在当前行上更新，无需更新索引

问题

  • 发现语句执行后即便没有插入数据表的自增主键还是进行了更新。
